Combine Technology Overlay, or CTO for short, originally created by Trudeau for the Clockwork framework, has been ported to Helix. This port brings all the original features of the plugin to Helix along with some minor optimizations to bring it up to today's standards.

Video Showcase

Players can use these commands to interact with this plugin

/CameraDisable

Remotely disable a Combine camera - IDs are shown on the HUD.

/CameraEnable

Remotely enable a Combine camera - IDs are shown on the HUD.

/CharSetBiosignalStatus

Turn a character's biosignal on or off. Will alert other units.

/CharSetHasTags

Set whether a Citizen has CID tags on their clothes.

/Request

Request assistance from Civil Protection.

/SetBiosignalStatus

Turn your biosignal on or off. Will alert all other units.

/SetSocioStatus

Update the sociostability status of the city.

Servers can customize usage of this plugin with these settings

Biosignal Distance (default: 0)

The maximum distance to see biosignals/cameras. Requests, losses, violations, etc are exempt. Set to 0 for infinite.

Biosignal Unobstruct (default: true)

Whether or not biosignals/cameras should be visible through walls. Requests, losses, violations, etc are exempt.

Citizen Distance (default: 2048)

The maximum distance to see movement violations and CID tags.

Expire Biosignals (default: 120)

Time in seconds until a lost biosignal disappears.

Expire Requests (default: 60)

Time in seconds until a request disappears.

Use Biosignal System (default: true)

Whether or not to use the biosignal system.

Use Tag System (default: true)

Whether or not to use the CID tag visor system.
