Hint System

by Riggs Mackay

  • Description
  • Options
  • License

Adds hints which might help you every now and then.

Players can customize their experience with these personal settings

Hints delay (default: 300)

The delay between hints.

Toggle hints (default: true)

Wether or not hints should be shown.

Toggle hints Sound (default: true)

Wether or not hints should play a sound.
